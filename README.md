# Lvory

极简跨平台客户端

## 功能特性

Lvory 是基于 Electron 的跨平台客户端。

## 快速开始

**系统要求**: Windows 10+  内核由 Lvory 自动管理。

**安装**: 从 GitHub Releases 下载最新版本。日常构建包含最新特性但可能不稳定，RC 版本经过简单测试，Stable 为正式版。

## 内核管理

Lvory 自动管理内核的下载与更新，也支持用户自行替换内核。

## 开发

构建编译参考 GitHub Actions 配置。
